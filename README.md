# WebGL Black Hole Simulation

A real-time WebGL simulation of a black hole.

---

## Demo
![Black Hole Preview](https://raw.githubusercontent.com/RandomUser15456/Webgl-Black-Hole-Simulation/main/preview.png)  
[Live Demo](https://zpayer.github.io/Webgl-Black-Hole-Simulation/)

---

## Features
- Draws black hole’s event horizon as a circle.
- Computes Schwarzschild radius: `r_s = 2 G M / c²`.
- Easy to extend with rays, disks, or animations.
